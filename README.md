# اختبارات محاكية نافس 2026 - Nafspro1

منصة اختبارات محاكية لاختبارات نافس الوطنية لطلاب المرحلة الابتدائية.

> **ملاحظة**: هذا المشروع عبارة عن قالب قابل لإعادة الاستخدام لإنشاء منصات اختبارات محاكية للمدارس المختلفة.

## المتطلبات

- Node.js (v18 أو أحدث)
- npm أو bun
- حساب Supabase

## التثبيت

```sh
# استنساخ المشروع
git clone https://github.com/iblwz/Nafspro1.git
cd Nafspro1

# تثبيت الاعتماديات
npm install

# إعداد المتغيرات البيئية (راجع SETUP.md للتفاصيل)
cp .env.example .env
# أضف بيانات Supabase الخاصة بك في ملف .env

# تشغيل خادم التطوير
npm run dev
```

## التقنيات المستخدمة

- Vite
- TypeScript
